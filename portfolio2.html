* Only designs which have actually shipped to customers, or which have been published on the web, in marketing materials, etc.

ClusterStor Manager (Seagate)

ClusterStor Manager is the 'single pane of glass' management software which ships with the ClusterStor Parallel Storage System by Seagate Technology.

Dashboard User Interfacemore_vert
Dashboard UI Design and Implementationclose

I worked with a small team of talented Architects, Engineers and Product Owner to design, implement and integrate this product UI and responsive dashboard into the ClusterStor Manager product. I created the logo, header image, overall layout, graphics, as well as information architecture and design of each individual view. We built it with custom HTML5 and CSS3 that I wrote along with Dojo JS and our custom backend framework.

Behind the scenes UX work included taking input directly from customers, creating user profiles, incorporating Accessibility features and progressive enhancements. Interaction Design work included sketches, wireframes, and functional prototypes.

The dashboard received rave reviews from customers and was featured at numerous SuperComputing conferences, which I also helped to present.

Sonexion System Manager (Cray)

Sonexion System Manger is monitoring and management software for the Sonexion Scale-Out Lustre Storage System.

Product Brandingmore_vert
Product Branding and Implementationclose

Sonexion System Manager is Cray's re-branded product based on ClusterStor Manager. With input from Cray, I designed and implemented the product logo, theme colors and overall look and feel. Cray supplied the 'blue swish' image in the middle of the header area as well as their official logo and asked that it be incorporated, which I did to produce the final design.

StorView (Xyratex)

StorView is enclosure management software from (the former) Xyratex, now owned by Seagate. I believe this product may be EOL (End of Life), although it still seems to be widely available online.

Modal UI Screensmore_vert
StorView Modal UI Screensclose

StorView was one of the first products (with a pre-existing UI) that I worked on after joining Xyratex. I designed graphical enclosure views, topological overlays, modal screen UIs, and icons.

Although StorView was re-designed and later versions were prototyped, these products did not end up being shipped, and so have not been included in this collection. It does account for the more dated appearance of some of the screens, however.
