ChannelCatCharters.com: Case Study

Project Profile

Type: Site Redesign

Category: Consumer-Facing Website

Description: The Channel Cat Charters website was launched in 2007. While it served them well for many years, it was built with yesterday's technology (Flash, Java applets) and was no longer working for them or many of their clients.

Purpose of the site:

  • For customers to see the grandeur of the ship and to contact the Charter Agent for a booking
  • For Charter Agent and other stakeholders to use the site as a sales and marketing tool

    • Goals for the redesign:

    • Make it easier for Users to navigate the site
    • Make it easy for customers to contact a Charter Agent
    • Virtual Tours are a big selling point, need to get those working again
    • Video interview with Extreme Yachts television show is a selling point, need it to work again

      • Problems to Solve:

      • Site too small on modern monitors, difficult to navigate as content is shon in tiny window
      • Home page animation not working on mobile (Flash-based)
      • Virtual Tours not working on mobile, or in some Desktop browsers (Java-applet based)
      • Gallery not working on mobile, difficult to navigate
      • Site required walk-through guidance to use
      • Directions need to be reworked, but cannot use physical address

      Technical Challenges:

      • Most Virtual Tour software tends to be Java based
      • Video content is embedded in Flash, only source available

      Preliminary Client notes:

      • Wants Big beautiful pictures
      • Believes "Less is more" with words, wants pictures to speak for themselves
      • Client is a Senior Citizen that struggles a bit with computers and technology
      • Client can send images via email, but many initial images sent are low-resoluton thumbnail images
      • Client is not interested in going the route of a booking and reservation system; as a high-end charterting business with an exclusive clientale, they hold the customers hand through every step of the booking and reservation process.
