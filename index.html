Design

User Experiences

Working with cross-functional Agile teams, my passion is for building great user experiences. This is achieved by first understanding business goals, customer needs, and user motivations. By utilizing proven human-centered design processes, rapid testing and iteration, we can deliver delightful and effective solutions.

User Interfaces

In an ever-changing world of devices, technologies and requirements it's essential to adapt and remain flexible on all fronts, from design and development processes to iteratively updated products. Responsive design and a mobile-first approach are my preferred techniques to designing relevant solutions.

Interactive Prototypes

After designing with tools such as Photoshop, Illustrator and Sketch, I code HTML5, CSS3, leverage popular frameworks like Bootstrap, libraries like jQuery, and version control with GIT; building flexible and robust prototypes with semantic code which can easily be changed or actively developed into shipping products.
